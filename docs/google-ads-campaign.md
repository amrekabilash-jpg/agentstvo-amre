## Чек-лист запуска (по порядку)

1. ☐ Создать GA4-ресурс → получить `G-XXXXXXXXXX`.
2. ☐ В Vercel добавить env `NEXT_PUBLIC_GA_ID=G-XXXXXXXXXX` → redeploy.
3. ☐ Проверить в GA4 Realtime: отправить тестовую форму → событие `generate_lead`; клик WhatsApp → `whatsapp_click`.
4. ☐ В GA4: Admin → Events → отметить оба события как Key events (конверсии).
5. ☐ Связать GA4 ↔ Google Ads (Admin → Product links).
6. ☐ В Google Ads: Цели → Конверсии → Импорт из GA4 → обе конверсии как Primary.
7. ☐ Активировать промокод $300.
8. ☐ Создать 3 кампании по этому документу.
9. ☐ Расширения: уточнения (Бесплатная консультация, Договор, Работаем по СНГ), структурированные описания, ссылки на /cases /contacts /blog, номер +7 701 728 22 36.
10. ☐ Через 7 дней: отчёт по поисковым запросам → добавить минусы.
