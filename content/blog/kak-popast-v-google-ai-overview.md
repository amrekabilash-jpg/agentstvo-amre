## Что такое Google AI Overview

С мая 2024 года Google запустил **AI Overview** — блок с AI-сгенерированным ответом, который появляется над всеми результатами поиска. По данным BrightEdge, он показывается в 84% информационных запросов.

Это революция для SEO: сайты теряют клики, но те, кого AI цитирует в своём ответе — получают **трафик нового качества**.

## Почему AEO важен для казахстанского бизнеса

Google AI Overview активен в Казахстане с осени 2024 года. Запросы типа:
- "лучшее агентство в Алматы"
- "как выбрать SEO-компанию"
- "стоимость разработки сайта в Казахстане"

...всё чаще получают AI-ответ на первом экране. Если ваш контент не в этом блоке — вы невидимы для значительной части аудитории.

## Как Google выбирает источники для AI Overview

Google использует несколько факторов:

**1. E-E-A-T (Опыт, Экспертность, Авторитетность, Доверие)**
- Реальный опыт автора (биография, кейсы)
- Уникальные данные и исследования
- Цитирование в других авторитетных источниках

**2. Структура контента**
- Чёткие ответы на вопросы в заголовках
- Списки, таблицы, схемы
- Логическая иерархия H1 → H2 → H3

**3. Технические факторы**
- Schema.org разметка (FAQPage, HowTo, Article)
- Core Web Vitals (скорость загрузки)
- HTTPS и безопасность

## 7 шагов для попадания в AI Overview

### Шаг 1: Исследуйте вопросы целевой аудитории

Используйте инструменты:
- **AnswerThePublic** — вопросы по ключевым словам
- **Google People Also Ask** — смежные вопросы
- **Яндекс.Вордстат** — для русскоязычной аудитории

Составьте список 20–50 конкретных вопросов вашей аудитории.

### Шаг 2: Создавайте "ответо-центричный" контент

Каждый раздел статьи — это вопрос + конкретный ответ в первых 2–3 предложениях.

**Плохо:**
> Оптимизация сайта включает множество аспектов, которые важно учитывать...

**Хорошо:**
> SEO-оптимизация сайта занимает 3–6 месяцев для заметных результатов. Это время необходимо для индексации, накопления ссылочной массы и анализа поведенческих факторов.

### Шаг 3: Внедрите FAQPage Schema

```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "Сколько стоит SEO в Алматы?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "SEO-оптимизация в Алматы стоит от 80 000 тенге в месяц для малого бизнеса. Полный пакет для среднего бизнеса — от 250 000 тенге."
    }
  }]
}
```

### Шаг 4: Используйте HowTo Schema для инструкций

Пошаговые руководства с HowTo Schema попадают в AI Overview значительно чаще обычных статей.

### Шаг 5: Оптимизируйте структуру страницы

- **H1**: один, содержит главный вопрос/тему
- **H2**: подвопросы и разделы
- **H3**: детали и примеры
- **Таблицы**: для сравнений и данных
- **Маркированные списки**: для перечислений

### Шаг 6: Улучшите Core Web Vitals

Google приоритизирует быстрые сайты. Цели:
- **LCP** (загрузка главного контента): < 2.5 сек
- **INP** (интерактивность): < 200 мс
- **CLS** (стабильность): < 0.1

### Шаг 7: Наращивайте E-E-A-T сигналы

- Добавьте биографию автора с реальными достижениями
- Публикуйте кейсы с конкретными цифрами
- Получайте упоминания на авторитетных ресурсах
- Проверьте наличие в Wikipedia, отраслевых каталогах

## Мониторинг результатов

Как отслеживать попадания в AI Overview:
1. **Google Search Console** → вкладка "Эффективность" → фильтр по типу запросов
2. **BrightEdge / Semrush** — специализированные AI-трекеры
3. **Ручная проверка** — регулярно проверяйте целевые запросы в режиме инкогнито

## Типичные ошибки при AEO

❌ Дублированный контент из других источников  
❌ Отсутствие конкретики (общие слова)  
❌ Нет даты публикации и обновления  
❌ Медленный сайт (> 4 сек загрузки)  
❌ Слабые E-E-A-T сигналы (анонимные авторы)

## Итог

Попасть в Google AI Overview реально, если:
- Контент отвечает на конкретные вопросы
- Правильно структурирован (заголовки + списки)
- Имеет техническую разметку Schema.org
- Сайт быстро загружается
