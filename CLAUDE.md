# Агентсво — Project Brain (Nick Saraev 2026)

## О проекте
Агентсво — современное цифровое агентство, которое помогает компаниям расти в B2B и B2C сегментах. Мы предлагаем три ключевых направления услуг.

## Направления услуг
1. **GEO / AEO / SEO-оптимизация** — поисковая и генеративная оптимизация
2. **Креативные услуги** — дизайн, брендинг, контент, видео
3. **Полная бизнес-автоматизация компаний** — AI-агенты, workflows, интеграции

## Стек технологий (по умолчанию)
- **Frontend:** Next.js 15 + Tailwind + shadcn/ui + Framer Motion
- **Backend:** Next.js App Router + Server Actions
- **Деплой:** Netlify / Modal
- **AI:** Claude Code + Antigravity + Skills

## Правила кода и дизайна (строго!)
- Всё на TypeScript
- Максимально чистый, современный, минималистичный дизайн
- Адаптивность mobile-first
- Использовать Screenshot Loop для референсов
- После каждого изменения: Prettier + git auto-commit

## Рабочий процесс (как учит Nick)
1. Plan Mode → проверка плана
2. Screenshot Loop или голосовой дамп
3. Создание / правки через Claude Code
4. Верификация результата (скриншоты + открытие в браузере)
5. Hooks: автоформатирование + git commit
