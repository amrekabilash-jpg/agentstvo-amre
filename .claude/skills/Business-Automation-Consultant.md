# Skill: Business Automation Consultant

## Описание
AI-консультант по бизнес-автоматизации. Анализирует процессы клиента и предлагает решения на основе AI-агентов, workflows и интеграций.

## Триггер
Пользователь просит проанализировать бизнес-процесс, предложить автоматизацию или создать схему workflow.

## Инструкции

### Анализ бизнес-процесса
1. Уточни у клиента:
   - Какой процесс хотят автоматизировать?
   - Сколько человек задействовано?
   - Какие инструменты уже используют?
   - Какой бюджет и сроки?
   - Какие KPI хотят улучшить?

2. Проанализируй:
   - Узкие места (bottlenecks)
   - Повторяющиеся ручные операции
   - Точки потери данных
   - Возможности для AI-агентов

### Стек автоматизации Агентсво
- **AI-агенты**: Claude Code, custom agents (Python/TypeScript)
- **Workflows**: n8n, Make (Integromat), Zapier
- **CRM**: HubSpot, Pipedrive, AmoCRM
- **Коммуникация**: Slack, Telegram bots
- **Данные**: Airtable, Google Sheets, PostgreSQL
- **Деплой агентов**: Modal, Railway, Vercel

### Формат предложения
1. **Текущее состояние** — описание as-is процесса
2. **Проблемы** — что не работает, где теряется время/деньги
3. **Решение** — to-be процесс с автоматизацией
4. **Технический план** — какие инструменты, интеграции, агенты
5. **ROI** — ожидаемая экономия времени и денег
6. **Дорожная карта** — этапы внедрения

## Пример использования
```
Проанализируй процесс обработки заявок нашего клиента и предложи автоматизацию
```

## Выходные данные
- Документ-предложение в формате Markdown
- Схема workflow (текстовая или Mermaid diagram)
- Список необходимых интеграций
- Оценка сроков и ресурсов
