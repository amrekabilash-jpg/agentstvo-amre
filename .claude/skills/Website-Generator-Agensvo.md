# Skill: Website Generator — Агентсво

## Описание
Генерация страниц и секций для сайта цифрового агентства "Агентсво". Использует фирменный стиль, стек и структуру проекта.

## Триггер
Пользователь просит создать страницу сайта, секцию, лендинг или любой UI для агентства.

## Инструкции

### Структура сайта Агентсво
Страницы:
1. **Главная** — Hero + Услуги + Кейсы + CTA + Футер
2. **Услуги** — Подробно о каждом направлении
3. **Кейсы / Портфолио** — Примеры работ
4. **О нас** — Команда, миссия, ценности
5. **Контакты** — Форма + карта + реквизиты
6. **Блог** — Статьи по SEO, автоматизации, дизайну

### Фирменный стиль
- Основной цвет: #0A0A0A (чёрный)
- Акцент: #6366F1 (индиго/фиолетовый)
- Фон: #FAFAFA (светло-серый) / #0A0A0A (тёмная тема)
- Шрифт: Inter
- Стиль: минималистичный, премиальный, технологичный

### При генерации страницы
1. Создай файл в `src/app/[route]/page.tsx`
2. Разбей на компоненты в `src/components/sections/`
3. Используй Server Components по умолчанию
4. Client Components только для интерактивности (формы, анимации)
5. Добавь метаданные (title, description) для SEO
6. Обеспечь полную адаптивность

### SEO-требования
- Semantic HTML (h1, h2, section, article, nav)
- Open Graph мета-теги
- JSON-LD структурированные данные
- Alt-тексты для изображений
- Корректная иерархия заголовков

## Пример использования
```
Создай главную страницу сайта Агентсво с hero, услугами и кейсами
```
