# Skill: Scrape Leads

## Описание
Автоматический парсинг и сбор лидов из открытых источников для B2B-продаж агентства.

## Триггер
Пользователь просит собрать лиды, найти потенциальных клиентов или спарсить контакты.

## Инструкции
1. Уточни у пользователя:
   - Целевую нишу / индустрию
   - Географию (город, страна)
   - Количество лидов
   - Какие данные нужны (email, телефон, сайт, соцсети)

2. Используй доступные инструменты:
   - Web Search для поиска компаний по нише
   - WebFetch для парсинга публичных каталогов и директорий
   - Bash для запуска скриптов парсинга (puppeteer, playwright)

3. Структурируй результат:
   - Создай CSV-файл с колонками: Company, Website, Email, Phone, City, Notes
   - Сохрани в `/data/leads/[дата]-[ниша].csv`

4. Валидация:
   - Проверь что email-адреса имеют корректный формат
   - Убери дубликаты
   - Отсортируй по релевантности

## Пример использования
```
Собери 50 лидов IT-компаний в Москве с email и сайтом
```

## Ограничения
- Только публичные данные
- Соблюдай robots.txt
- Не парси персональные данные без согласия
