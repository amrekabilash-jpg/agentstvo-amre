# Skill: UI Design Master (Screenshot Loop)

## Описание
Создание и итерация UI-дизайна через Screenshot Loop — методологию Nick Saraev для точного воспроизведения дизайн-референсов.

## Триггер
Пользователь просит создать UI, дизайн страницы, компонент или хочет итерировать дизайн по скриншоту.

## Инструкции

### Screenshot Loop (цикл)
1. **Получи референс**: Попроси пользователя предоставить скриншот или URL референса
2. **Анализируй**: Изучи скриншот — цвета, типографику, отступы, компоновку, стиль
3. **Реализуй**: Создай компонент на React + Tailwind + shadcn/ui + Framer Motion
4. **Сделай скриншот**: Открой результат в браузере, сделай скриншот
5. **Сравни**: Сравни результат с референсом
6. **Итерируй**: Повтори шаги 3-5 пока результат не совпадёт с референсом

### Принципы дизайна
- Mobile-first всегда
- Минимализм — меньше элементов, больше воздуха
- Консистентность — единый стиль во всём проекте
- Анимации через Framer Motion (subtle, не перегружать)
- Цветовая палитра — максимум 3-4 основных цвета

### Стек компонентов
- Layout: Tailwind CSS Grid / Flexbox
- UI Kit: shadcn/ui (Button, Card, Dialog, etc.)
- Анимации: Framer Motion (fade, slide, scale)
- Иконки: Lucide React
- Шрифты: Inter / системные

## Пример использования
```
Создай hero-секцию как на этом скриншоте [прикрепить изображение]
```

## Выходные данные
- React TSX компонент в `src/components/`
- Tailwind стили inline
- Responsive (sm, md, lg, xl)
